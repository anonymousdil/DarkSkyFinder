AQI Service Integration Test

⚠️ Important: Manual Browser Testing Only

This test file is designed for manual testing in a web browser only.

It makes direct API calls to AQICN, which will be blocked in GitHub Actions CI environment.

How to use:

  1. Make sure you have configured VITE_AQICN_TOKEN in your .env file
  2. Start the development server: npm run dev
  3. Open this file directly in your browser: http://localhost:5173/test-aqi-service.html
  4. Click "Run All Tests" to verify the AQI integration

Note: This file should NOT be run as part of automated CI/CD pipelines.

Test Configuration

  • Rural Australia (-25.2744°S, 133.7751°E) - Expected: Good AQI (0-100)
Note: AQICN returns AQI values on the US EPA 0-500 scale directly.
